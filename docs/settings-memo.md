# 開発環境構築用のメモの最新版

## インストールが必要なアプリ

- chrome
- chrome のアカウント設定
- vscode
- google 日本語入力
- vscode の設定
  - settings.json と keybind.json をセットした
- chrome beta
- Rectangle
- clipy
- dropbox
- terminal の iceburg テーマ
- homebrew
  - homebrew インストール中に xcode command line tools がインストール
  - brew install bash-completion
  - brew install git
  - 上記の completion の設定を bashrc に書き、.inputrc を追加
- git config 設定
  - .gitconfig
- github desktop
- miniforge (for python)
- scroll reverser
- vscode の拡張機能
  - eslint
  - live server
  - git graph
  - gitlens
  - japanese language pack
  - path intellisense
  - copilot
  - prettier
  - package manager
  - sublime text keymap and settings importer
- coteditor
- soucetree
- karabiner-element
- quick shade
- notion
- obsidian
- antigravity
- その他ブラウザ(brave, edge)

## pc 設定

- dock の表示速度の設定
  - `defaults write com.apple.dock autohide-time-modifier -int 0; killall Dock`
